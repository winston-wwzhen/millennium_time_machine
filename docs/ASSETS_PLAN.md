### 加载策略

#### 1. 首屏加载（本地）
```
用户打开小程序
    ↓
立即显示：本地图标 + UI框架
    ↓
后台加载：云存储素材（壁纸、音效等）
```

#### 2. 按需加载（云存储）
```javascript
// 使用素材时再从云端获取
async loadLionDance() {
  const urls = await Promise.all([
    getTempFileURL('lion/lion_dance_1.png'),
    getTempFileURL('lion/lion_dance_2.png'),
    getTempFileURL('lion/lion_dance_3.png'),
  ])
  // 下载并缓存
  return urls
}
```

#### 3. 预加载关键资源
```javascript
// 首次进入时预加载桌面壁纸
onLoad() {
  this.preloadWallpaper()
}
```

### 图片优化建议

| 优化方式 | 说明 | 效果 |
|----------|------|------|
| **压缩图片** | TinyPNG等工具 | 减少30-50% |
| **使用WebP** | 更小体积 | 减少25-35% |
| **裁剪尺寸** | 不超过显示尺寸 | 按需调整 |
| **合并图片** | 雪碧图技术 | 减少请求 |

---

### 缓存策略

```javascript
// 下载后缓存到本地
wx.downloadFile({
  url: cloudURL,
  success: (res) => {
    // 保存到本地
    wx.saveFile({
      tempFilePath: res.tempFilePath,
      success: (saveRes) => {
        // 下次直接用本地缓存
        wx.setStorageSync('cached_wallpaper', saveRes.savedFilePath)
      }
    })
  }
})
```

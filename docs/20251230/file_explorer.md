```
┌─────────────────────────────────────────────┐
│ 文件 | 编辑 | 查看 | 收藏 | 帮助            │
```

## 隐藏文件实现方案

### 状态存储结构

```javascript
fileViewOptions: {
  'C:\\': { showHidden: false },
  'C:\\Windows': { showHidden: false },
  'C:\\Windows\\Temp': { showHidden: false },
  'USB:\\': { showHidden: false },
  'USB:\\私密文件夹': { showHidden: false },
  // ...
}
```

### 隐藏文件定义

- 文件名以 `.` 开头（如 `.`）
- 或文件数据中有 `hidden: true` 属性

### 交互逻辑

- 默认：隐藏文件不显示
- 点击"显示所有文件" → 隐藏文件出现（图标半透明 opacity: 0.6）
- 按目录独立控制状态

## 线索文件（非彩蛋，无奖励）

| 位置 | 类型 | 说明 |
|------|------|------|
| config.ini 末尾 | 隐藏段落注释 | 指向deep目录 |
| system.log 时间戳 | 异常时间+路径 | 指向deep\0xFFFF\ |
| win.ini 末尾 | 隐藏歌词注释 | 指向help.txt |
| thumbs.db | 隐藏文件内容 | AI日记片段 |
| USB\私密文件夹\secret_note.txt | 文件内容 | AI线索 |
| USB\私密文件夹\.hidden路径 | 目录结构 | 指向deep |
| USB\私密文件夹\乱码.txt | 摩斯密码 | 需解密 |
| USB\私密文件夹\程序员笔记.txt | 时间戳异常 | 2006-02-30 25:61 |

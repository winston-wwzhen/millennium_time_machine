# USB盘完整设计

## 安装逻辑（需先安装 usbstor.sys 驱动）

### 安装前

- USB盘图标：灰色/禁用状态
- 点击提示："未安装USB存储设备驱动，请前往 C:\Windows\System32\Drivers 安装 usbstor.sys"

### 安装过程

1. 打开 C:\Windows\System32\Drivers
2. 点击 usbstor.sys
3. 确认安装驱动

### 安装后

- USB盘图标：正常状态
- 点击打开文件浏览器

## 一、零预算运营策略

### 1.1 核心策略

```
阶段目标：积累用户 + 验证需求
         ↓
核心策略：先免费，不做硬变现
         ↓
重点关注：用户留存 > 收入
```

### 1.2 运营渠道

| 平台 | 用途 | 内容形式 | 频率 |
|------|------|---------|------|
| **微信公众号** | 主阵地、私域流量 | 图文 | 每日 |
| **小红书** | 内容种草 | 图文 | 每日 |
| **微博** | 传播扩散 | 图文 | 每日 |
| **朋友圈** | 私域传播 | 海报 | 每日 |

### 1.3 内容策略

**不做产品预热，做内容预热**

| 可以展示 | 不要展示 |
|---------|---------|
| 怀旧情感、年代回忆 | 具体功能截图 |
| 非主流文化、QQ空间 | 产品细节 |
| "能回到2005年" | 技术实现方案 |
| 情感共鸣内容 | 数据结构、事件列表 |

| 素材 | 风险 | 替代方案 |

| Windows 98桌面壁纸 | 微软版权 | ✅ AI生成"类似风格" |

### 4.4 AI生成提示词规范

**安全做法**：
- ✅ "复古电脑桌面，90年代风格"

- ❌ "Windows 98桌面"（含品牌）
- ❌ "微软风格图标"（含品牌）

## 五、盈利模式分析

### 5.1 个人主体的限制

| 限制 | 影响 |
|------|------|
| ❌ 不能微信支付 | 无法直接变现 |
| ❌ 不能做付费会员 | 无法订阅制 |
| ❌ 不能做内购 | 无法卖虚拟物品 |

**结论**：个人主体只能做"免费+广告"或"导流"模式。

### 5.2 短期：个人主体阶段

#### 广告变现（主）

| 形式 | 位置 | 预估eCPM |
|------|------|---------|
| 激励视频 | 看广告获得额外经验/Q点 | 50-100元 |
| Banner | 桌面底部、页面底部 | 10-30元 |
| 原生广告 | 桌面图标 | 20-50元 |

**注意**：
- 广告会影响怀旧体验，需谨慎
- 建议只用激励视频（用户主动选择）
- 目标：覆盖AI成本即可

#### 导流变现

| 模式 | 说明 |
|------|------|
| 公众号导流 | 桌面放"关注公众号"图标 |
| 社群导流 | 引流到微信群 |
| 其他小程序 | 导流到你的其他小程序 |

### 5.3 中期：企业主体升级

```
个人主体 → 注册个体户/公司 → 升级为企业主体
```

**解锁能力**：
- ✅ 微信支付
- ✅ 付费会员（如"黄钻会员"）
- ✅ 虚拟商品（Q点充值、皮肤购买）
- ✅ 更多类目

**企业主体的变现方式**：

| 模式 | 定价 | 说明 |
|------|------|------|
| 会员订阅 | 9.9元/月 | 去除广告+特权 |
| 虚拟货币 | 1元=100Q点 | 购买虚拟物品 |
| 装扮皮肤 | 1-5元/个 | 非主流头像、空间皮肤 |
| 去广告 | 6元一次性 | 永久去广告 |

### 5.4 长期：IP/周边

| 模式 | 说明 |
|------|------|
| 周边商品 | 非主流风格T恤、手机壳、贴纸 |
| 授权联名 | 与怀旧品牌联名 |
| 内容付费 | 怀旧电子书、播客 |

### 5.5 成本与收入

**成本估算**：

| 成本项 | 月成本 | 说明 |
|--------|--------|------|
| AI API调用 | 100-500元 | 取决于用户量 |
| 云服务器 | 0-100元 | 微信云开发免费额度够用 |
| **总计** | **100-650元** | |

**收入预估**：

| 用户规模 | 激励视频收入（月） |
|---------|------------------|
| 1000 DAU | 50-200元 |
| 5000 DAU | 200-1000元 |
| 10000 DAU | 500-2000元 |

### 5.6 变现建议

**当前阶段建议**：
1. 先做用户，后想变现 - 用户是基础
2. 轻量广告 - 只用激励视频，用户主动选择
3. 导流私域 - 公众号、社群是核心资产
4. 控制成本 - AI调用要控制频率

### 6.6 注意事项

**合规注意**：

| 注意 | 说明 |
|------|------|
| 隐私保护 | 必须脱敏用户信息 |
| 用户授权 | 需要用户同意参与排行榜 |

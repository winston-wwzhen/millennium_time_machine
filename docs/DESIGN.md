## 一、产品定位与分析

### 1.1 核心定位

```
千禧时光机 = 怀旧 + AI + 工具
            ↓
    可交互的怀旧体验
    而非静态内容展示
```

### 1.2 目标用户

- **主要**：80后、90后、00后
- **次要**：对怀旧文化、非主流文化感兴趣的群体
- **核心**：经历过2005-2025年的中国互联网用户

### 1.3 产品价值

| 层次 | 价值 |
|------|------|
| 功能层 | AI聊天、人生模拟、怀旧工具 |
| 情感层 | 青春回忆、怀旧共鸣 |
| 社交层 | AI模拟社交、情感陪伴 |

### 1.4 竞争优势

| 优势 | 说明 |
|------|------|
| **情感共鸣强** | 2005-2025真实历史事件，覆盖80/90/00后青春记忆 |
| **差异化明显** | Win98+非主流+AI聊天，市场上没有完全一样的 |
| **内容深度** | 36种AI模式、100个AI好友、6个群聊、50+历史事件 |
| **用户粘性** | 多元功能（聊天、游戏、人生模拟、工具），用户停留时间长 |
| **合规意识** | 用AI模拟社交，规避了个人主体小程序限制 |

## 二、如果当时.exe - 人生模拟设计

### 2.1 游戏概述

**定位**：2005-2025中国历史人生模拟器

**核心玩法**：
- 用户选择出生年份（默认2005）、性别
- 从2005年开始，经历20年时光
- 每年遇到真实历史事件，做出选择
- 选择影响属性（科技、财富、社交、健康、幸福）
- 最终达成不同的"2025人生结局"

### 2.2 属性系统

| 属性 | 说明 | 初始值 | 范围 |
|------|------|--------|------|
| 科技 | 对互联网、新技术的接受和使用能力 | 50 | 0-100 |
| 财富 | 经济水平和消费能力 | 50 | 0-100 |
| 社交 | 人际关系、沟通能力 | 50 | 0-100 |
| 健康 | 身体和心理状况 | 80 | 0-100 |
| 幸福 | 综合幸福指数 | 50 | 0-100 |

### 2.3 事件系统

**事件数量**：50+个真实历史事件

**事件分类**：
| 分类 | 数量 | 示例 |
|------|------|------|
| 科技事件 | 15个 | 2008北京奥运会、4G网络普及 |
| 财富事件 | 10个 | 2008金融危机、电商兴起 |
| 社交事件 | 10个 | 微信诞生、短视频时代 |
| 健康事件 | 10个 | 疫情、健康意识提升 |
| 幸福事件 | 10个 | 毕业求职、结婚生子 |

### 2.4 结局系统

**结局类型**：
| 类型 | 条件 | 示例 |
|------|------|------|
| 普通结局 | 各属性均衡 | "平凡而幸福" |
| 专业结局 | 某属性突出 | "互联网大厂CEO" |
| 稀有结局 | 特殊选择组合 | "穿越者" |
| 隐藏结局 | 特定条件触发 | "彩蛋大师" |

### 2.5 复玩价值

- **不同选择**：每年都有2-4个选择，可产生不同结果
- **不同年份**：可选择不同出生年份，经历不同事件
- **收集结局**：多种结局待解锁
- **分享传播**：结局可生成海报分享

## 四、彩蛋系统设计

### 4.1 设计理念

```
所有可见按钮都可点击 + 大量隐藏彩蛋
            ↓
     用户探索 → 发现彩蛋 → 惊喜感
            ↓
     激发更多探索兴趣 → 分享传播
```

### 4.2 彩蛋分类

| 稀有度 | 数量 | 奖励（时光币） | 示例 |
|--------|------|--------------|------|
| 普通 | 7个 | 400-1000 | 小狮子跳舞、小狮子说话 |
| 稀有 | 3个 | 1500-2000 | 蓝屏噩梦、隐藏图标 |
| 史诗 | 1个 | 5000 | 深夜党专属 |
| 传说 | 1个 | 10000 | Konami秘籍代码 |

### 4.3 彩蛋列表

**普通彩蛋**（400-1000 时光币）：
- 🦁 舞动的小狮子 - 点击小狮子10次
- 💬 它会说话 - 长按小狮子
- 📂 底部秘密 - 点击任务栏10次
- 🎨 换了个心情 - 双击桌面背景切换
- 🗑️ 垃圾清理员 - 点击回收站
- 💻 电脑专家 - 点击我的电脑C盘
- 🌐 网瘾少年 - 点击浏览器图标

**稀有彩蛋**（1500-2000 时光币）：
- 💻 那个年代的噩梦 - 点击桌面空白处50次（蓝屏）
- 👁️ 消失的角落 - 点击桌面右下角出现隐藏图标
- ⏰ 特殊时刻 - 特定时间进入（如0点、12点）

**史诗彩蛋**（5000 时光币）：
- 🌙 深夜党专属 - 凌晨0点-1点进入桌面

**传说彩蛋**（10000 时光币）：
- 🎮 传说中的秘籍 - 输入Konami代码（↑↑↓↓←→←→BA）

### 4.4 彩蛋发现系统

**小程序内彩蛋图鉴入口**：设置 → 彩蛋图鉴

**浏览器内彩蛋秘籍网站**：
- 入口：浏览器图标 → 地址栏输入 "eggs.2005.com"
- 显示：所有已发现彩蛋列表、发现进度、提示信息

### 4.5 开发原则

| 原则 | 说明 |
|------|------|
| **不影响主流程** | 不影响用户正常使用 |
| **有怀旧感** | 彩蛋本身也有年代感 |
| **有惊喜感** | 发现彩蛋时要有"哇"的感觉 |
| **可传播** | 有些彩蛋值得分享 |
| **适度数量** | 不宜太多，保持神秘感 |
| **持续更新** | 定期添加新彩蛋 |

## 五、分享裂变系统设计

### 5.1 设计理念

```
用户获得"值得分享"的内容
        ↓
    便捷的分享入口
        ↓
    一键生成分享内容
        ↓
    好友看到 → 进入小程序
        ↓
    新用户被吸引 → 也想分享
        ↓
        裂变增长
```

### 5.2 可分享内容

| 内容类型 | 优先级 | 说明 |
|---------|--------|------|
| 人生结局海报 | 🔴 高 | 最有分享价值 |
| 非主流头像 | 🟡 中 | 视觉冲击力强 |
| AI聊天截图 | 🟡 中 | 内容有趣 |
| 等级提升 | 🟡 中 | 社交货币 |
| 彩蛋发现 | 🟡 中 | 激发探索欲 |
| 踩一踩 | 🟢 低 | 互动性强 |

### 5.3 分享激励

| 激励类型 | 内容 | 触发条件 |
|---------|------|----------|
| 首次分享 | 100 Q点 | 第一次分享 |
| 每日分享 | 50 Q点 | 每天第一次分享 |
| 分享回流 | 200 Q点 | 有好友通过分享进入 |
| 彩蛋分享 | 50 Q点 | 分享发现的彩蛋 |
| 成就分享 | 100 Q点 | 分享稀有成就 |

### 5.4 分享入口

| 位置 | 触发时机 | 分享内容 |
|------|---------|---------|
| 结局页面 | 完成游戏后 | 人生结局海报 |
| 成就达成 | 获得成就时 | 成就分享卡片 |
| 等级提升 | 升级时 | 等级提升卡片 |
| 彩蛋发现 | 发现彩蛋后 | 彩蛋发现卡片 |
| 头像生成 | 生成头像后 | 头像分享卡片 |
| AI聊天页 | 聊天页按钮 | AI对话分享 |
| 踩一踩后 | 踩完后 | 空间分享卡片 |

## 六、CDK系统设计

### 6.1 系统概述

```
你在私域（公众号/社群）售卖CDK
        ↓
用户购买后获得CDK码
        ↓
在小程序内输入CDK激活
        ↓
解锁新应用/新功能
```

### 6.2 CDK类型

| 类型 | 用途 | 定价 |
|------|------|------|
| 应用激活码 | 激活新应用 | 9.9-19.9元 |
| 功能解锁码 | 解锁特殊功能 | 4.9-9.9元 |
| 特权码 | 去广告+特权 | 19.9-99.9元 |
| 礼包码 | Q点+装饰+称号 | 4.9-9.9元 |
| 限时码 | 节日/活动限定 | 变动 |

### 6.3 CDK格式

```
格式：XXXX-XXXX-XXXX-XXXX
示例：MILL-2005-NOST-ALGI
```

**前缀含义**：
| 前缀 | 说明 |
|------|------|
| MILL | 千禧时光机基础 |
| IFTH | 如果当时.exe |
| QCIO | QCIO高级版 |
| VIP | 特权会员 |
| THEME | 主题 |
| GIFT | 礼包 |

### 6.4 防滥用设计

| 限制 | 说明 |
|------|------|
| 一人一码 | 每个CDK只能绑定一个用户 |
| 使用时效 | 激活后7天内必须使用 |
| 设备绑定 | 激活后绑定设备，不可转移 |
| 有效期 | CDK有有效期（通常1年）|

## 七、QCIO社交系统设计

### 7.1 产品概述

QCIO 是"千禧时光机"小程序内的核心虚拟社交模块，致敬 2005-2008 年代的经典即时通讯软件 QQ 的 UI 与交互。

**定位**：怀旧模拟器（非真实社交软件）

### 7.2 核心功能

#### 7.2.1 用户身份面板

| 功能 | 说明 |
|------|------|
| 头像系统 | 默认👤，可点击编辑昵称和签名 |
| QCIO号码 | 5位数字ID (10000-99999)，随机生成 |
| 个性签名 | 默认火星文风格 |
| 等级系统 | ⭐→🌙→☀️→👑→💎 |

#### 7.2.2 好友列表

- **AI好友数量**：普通用户20个，VIP用户100个
- **分组结构**：12个分组，100个AI网友
- **聊天模式**：36种AI人设（qingwu、sadsoul、netadmin等）
- **群聊**：6个群聊，每个18-24人

**分组列表**：
| 分组 | 人数 | 风格 |
|------|------|------|
| 葬爱家族 | 22人 | 非主流火星文 |
| 网游开黑 | 6人 | 游戏爱好者 |
| 网吧常驻 | 6人 | 技术、网吧 |
| 网络评论家 | 7人 | 键盘侠、吐槽 |
| 神秘人士 | 9人 | 玄学、中二 |
| 情感咨询 | 6人 | 恋爱、八卦 |
| 学霸联盟 | 7人 | 学习、技术 |
| 文艺青年 | 8人 | 诗歌、古风 |
| 娱乐达人 | 7人 | 电影、音乐 |
| 佛系一族 | 5人 | 禅意、极简 |
| 普通人 | 7人 | 日常生活 |
| 工具人 | 10人 | 转换功能 |

#### 7.2.3 QQ空间

**主页**：
- 每日签到
- QCIO农场入口

**日志**：
- 心情类型：💔伤感、🔥热情、💕甜蜜、🌫️迷茫
- AI自动生成火星文风格日志
- 访问量和点赞数统计

**留言**：
- 踩一踩功能
- 自动生成留言
- 最近访客记录

#### 7.2.4 QCIO农场

**双轨种植**：
- 传统作物：小麦、玉米、番茄、南瓜、草莓等（周期短）
- 心情作物：忧伤.exe、寂寞.bat、初恋.dll等（周期长，高收益）

**土地系统**：
- 初始6块，最多24块
- 农场等级升级解锁更多土地

**农场等级**：
| 等级 | 所需经验 | 解锁土地 |
|------|----------|----------|
| Lv.1 | 0 | 6块 |
| Lv.3 | 300 | 9块 |
| Lv.10 | 3000 | 15块 |
| Lv.30 | 50000 | 24块 |

| qcio_users | QCIO账号信息 |

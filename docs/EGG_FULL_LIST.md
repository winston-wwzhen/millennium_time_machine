## 技术实现

### 核心文件

| 文件 | 说明 |
|------|------|
| `egg-system.js` | 彩蛋系统核心，所有彩蛋配置和状态管理 |
| `pages/index/index.js` | 桌面彩蛋触发 |
| `pages/browser/index.js` | 浏览器工具彩蛋 |
| `pages/avatar/index.js` | 非主流相机彩蛋 |
| `components/my-computer/index.js` | 我的电脑彩蛋（C/D/USB盘探索） |

### 彩蛋数据结构

```javascript
{
  id: 'egg_id',           // 唯一标识
  name: '彩蛋名称',
  description: '描述文本',
  hint: '触发提示',
  rarity: 'common',       // common, rare, epic, legendary
  type: 'click',          // click, longpress, time, sequence, action
  reward: {
    coins: 1000,          // 时光币奖励
    badge: '徽章名称'
  }
}
```

# 千禧时光机 - 产品功能说明书

## 1. 产品概述

千禧时光机是一个充满怀旧风格的微信小程序，提供各种文字处理工具和娱乐功能，让用户重温Windows 98时代的经典界面和交互体验。

**产品定位**：复古风格工具集合 + 怀旧游戏体验

## 2. 核心功能模块

### 2.1 火星文翻译器 (pages/mars)
**功能描述**：在火星文、正常中文、英文之间相互转换

**主要特性**：
- 智能三向翻译（中文 ↔ 火星文 ↔ 英文）
- 实时转换预览
- 一键复制结果
- 转换历史记录

**技术实现**：
- 前端：字符映射表转换
- AI对话：集成ChatGPT/GLM模型

### 2.2 非主流头像生成器 (pages/avatar)
**功能描述**：生成经典非主流风格的头像

**主要特性**：
- 多种复古滤镜效果
- 像素化处理
- 一键保存到相册
- 无需登录即可使用

### 2.3 AI聊天对话 (pages/chat)
**功能描述**：与AI助手进行智能对话

**主要特性**：
- 多轮对话支持
- 聊天记录保存
- 流畅的对话体验
- 复古聊天窗口UI

### 2.4 通用翻译器 (pages/translator)
**功能描述**：提供多语言翻译功能

**主要特性**：
- 支持多种语言对
- 简洁易用的界面
- 实时翻译显示

### 2.5 星域探险 (pages/star-explorer)
**功能描述**：抽象主题的扫雷游戏

**游戏特性**：
- 三种难度等级：
  - 新手探索者（9×9，10个雷）
  - 资深领航员（16×16，40个雷）
  - 传奇指挥官（30×16，99个雷）
- 深空主题UI设计
- 最佳时间记录
- 递归式揭开逻辑
- 智能标记剩余地雷
- 炫酷的动画效果

### 2.6 千禧浏览器 (pages/browser)
**功能描述**：复古IE浏览器界面

**主要特性**：
- 经典Windows 98风格工具栏
- 支持网页浏览功能
- 怀旧的浏览器UI
- 地址栏导航

### 2.7 农场游戏组件 (components/farm-game)
**功能描述**：可复用的农场游戏组件

**组件特性**：
- 种植系统
- 收获机制
- 倒计时功能
- 可集成到其他页面

## 3. 界面设计

### 3.1 桌面系统（首页）
**设计理念**：模拟Windows 98桌面

**主要元素**：
- 经典桌面图标布局
- 开始菜单
- 任务栏
- 系统托盘
- 桌面小部件（便签、小狮子助手）

### 3.2 视觉风格
- 配色：经典灰色主题
- 字体：系统默认字体
- 边框：立体3D效果（outset/inset）
- 图标：16×16像素风格

## 4. 技术架构

### 4.1 前端技术栈
- **框架**：微信小程序原生框架
- **语言**：JavaScript
- **样式**：WXSS
- **组件**：自定义组件支持

### 4.2 后端服务
- **云服务**：微信云开发
- **数据库**：云数据库
- **云函数**：
  - chat：AI对话服务
  - users：用户数据管理（预留）
  - mood_logic：情绪逻辑处理（预留）

### 4.3 项目结构
```

├── app.js              # 小程序入口
├── app.json            # 全局配置
├── pages/              # 页面目录
```

## 5. 数据存储

### 5.1 本地存储
- 用户偏好设置
- 游戏最高分记录
- 浏览历史

### 5.2 云数据库（预留）
- users：用户信息
- chat_history：聊天记录
- game_scores：游戏分数排行

## 6. 合规性说明

由于项目采用了以下策略，确保完全符合微信小程序规范：

1. **无需登录系统**：所有功能均可直接使用，避免收集用户信息
2. **纯工具属性**：专注于提供实用工具和游戏，无社交功能
3. **内容安全**：
   - AI对话内容通过云函数安全过滤
   - 图片处理使用微信官方安全接口
   - 无用户发布内容功能

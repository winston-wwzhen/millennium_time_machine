### 文章格式

每篇文章包含：
- 📸 配图方案总览
- 🎨 封面图 Prompt
- 正文（含【插入配图】标记）
- 📋 配图使用说明
- 🎯 文章结构

### 使用方式

1. **生成配图**：复制文章中的 Prompt 到 AI 绘画工具
2. **编辑文章**：使用秀米/135编辑器排版
3. **插入配图**：按照【插入配图】标记插入对应图片
4. **发布**：设置发布时间

## 🎨 配图通用说明

### 规避版权风险

所有 Prompt 已遵循：
- ❌ 不使用 QQ 企鹅（腾讯商标）
- ❌ 不直接复制 Windows 界面
- ❌ 不使用任何注册商标元素
- ✅ 使用通用复古风格
- ✅ 原创像素/矢量插画风格

### 推荐工具

- **Midjourney**：艺术效果最好，付费
- **Stable Diffusion**：免费开源，需本地部署
- **DALL-E 3**：ChatGPT Plus 可用
- **文心一格**：百度出品，中文友好
- **通义万相**：阿里出品，免费额度

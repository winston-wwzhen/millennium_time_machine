# 数据库初始化脚本

本目录存放各数据库表的初始化数据，用于环境部署或数据迁移。

## 目录结构

```
db-init/
├── README.md                    # 本说明文件
├── qcio_ai_contacts/            # AI 好友列表表
│   └── ai_contacts_import.json
└── [其他表]/
    └── ...
```

## 使用方法

### 微信开发者工具导入

1. 打开微信开发者工具
2. 进入「云开发」→「数据库」
3. 选择对应的集合（如 `qcio_ai_contacts`）
4. 点击「导入」按钮
5. 选择对应的 `.jsonl` 文件
6. 确认导入

## 文件格式说明

所有导入文件使用 **JSONL (JSON Lines)** 格式：
- 每行一个独立的 JSON 对象
- 不是标准的 JSON 数组格式
- 微信云数据库要求此格式

## 注意事项

- 导入前请确认目标集合已创建
- 导入时会覆盖相同 `_id` 的记录
- 建议先在测试环境验证
